Calculadora de Gordura Corporal

Projeto em Python que calcula a gordura corporal do usuário utilizando a fórmula da Marinha Americana.

Tecnologias utilizadas:
Python
Biblioteca Math

Como executar:
Instale o Python e execute o arquivo principal do projeto.

python app.py

Objetivo:
Projeto criado para prática de funções, bibliotecas, laços de repetição e organização de código.
